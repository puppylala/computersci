```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        dict1={}
        list1=[]
        for i in strs:
            list2=list(i)
            list2.sort()
            dict1[tuple(list2)]=[]
        for k in strs:
            m=list(k)
            m.sort()
            dict1[tuple(m)].append(k)
        return list(dict1.values())
```
题目：给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

示例 1:

输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]

输出: [["bat"],["nat","tan"],["ate","eat","tea"]]

解释：

在 strs 中没有字符串可以通过重新排列来形成 "bat"。
字符串 "nat" 和 "tan" 是字母异位词，因为它们可以重新排列以形成彼此。
字符串 "ate" ，"eat" 和 "tea" 是字母异位词，因为它们可以重新排列以形成彼此。
# 思考
题目难点在于字母的顺序可以任意变换，但是又不能直接用冻结集合frozenset作为字典键，比如dddggg和ddddggg集合中是一样的，但是他们并不可以合并在一起。python标准库中没有只保留无序性而没有保留元素不可重复性的集合。
最后选择了对列表比较大小的方式，比较大小是依据字典序按照元素顺序比较。
