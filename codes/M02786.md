```python
a=int(input())  
list1=[]  
for i in range(a):  
    b=int(input())  
    def mol(b):  
        if b==1:  
            return 1  
        if b==2:  
            return 2  
        if b>=3:  
            return 2*mol(b-1)+mol(b-2)  
    list1.append(mol(b)%32767)  
for i in list1:  
    print(i)  
#这是原始递归。内存不会超，时间会超，因为从n算回到1，还要从1累积至n
```
